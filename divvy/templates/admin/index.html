This is the administrative panel for Divvy. It allows performing maintenance and configuration tasks.

The Curator tab has an editable list of all the curators on the team. In order for a team member to be considered by Divvy, she has to be listed. The list is editable. Checker marks whether a curator participates in QA.

The Folder tab lists paths to all the folders Divvy is searching for files to check in. The paths reflect the path mapping on the server! The list is editable.

The File tab lists files currently in Divvy's database. This does not need to be touched normally.
